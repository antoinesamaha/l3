This is a list of the main keyboard shortcuts available through most of your 01Barmaja application:

·  [DEL] : The delete key can be used to delete the current item from a list

·  [INS] : The insert key can be used to create a new item in a list

·  [SPACE] : The space bar key can be used to edit the current item in a list

·  [ESCAPE] : The escape key cancels the current modification and goes back to the previous screen

·  [CTRL-ENTER] : The control enter key allows validating or saving and existing the current view. It equivalent to pressing the validation button in the bottom of the screen

·  [ALT-F] : Allows to popup the filter definition of the list
